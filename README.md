# J Intelligenza Website

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm install
npm run build
```

## Deploy to Vercel

1. Create a new GitHub repository and upload all files in this folder.
2. Sign in to Vercel.
3. Import the GitHub repository.
4. Vercel should detect Vite automatically.
5. Click Deploy.
6. In Vercel project settings, add your domain: `jintelligenza.com`.
7. Add the DNS records that Vercel shows at your domain registrar.

## Before going live

- Replace the placeholder phone number in `src/App.jsx`
- Keep `public/logo.png` as your official website logo
